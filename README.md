# #38 Simple Words

**"Simple Words"** is a minimalist web tool designed to bridge the gap between complex language and clear communication. It instantly converts difficult terminology or "Big Words" into plain, easy-to-understand language suitable for a 12-year-old.

Perfect for creators, writers, and professionals who want their message to reach a global audience without the "stiffness" of formal jargon.

---

### **Core Philosophy**
* **1 Tool / 1 Action:** Paste your text, make it simple. That’s it.
* **Bi-lingual Power:** Supports both **English** (removes academic/corporate jargon) and **Japanese** (converts difficult Kanji/jargon into plain speech).
* **Live Editing:** The result area acts as a "scratchpad"—tweak the text on the fly before copying.

---

### **Key Features**
* **Highlight Insight:** Automatically highlights transformed words so you can see exactly where the "thorns" were removed.
* **Instant Clear & Copy:** Minimalist buttons for a frictionless workflow.
* **Privacy First:** Browser-only processing. Your text never leaves your device.
* **Zero Bloat:** Lightweight code optimized for low-spec mobile and desktop browsers.

---

### **How to Use**
1.  Select your language (**JP** or **EN**).
2.  Paste your "thorny" text into the input field.
3.  Click **[ MAKE IT SIMPLE ]**.
4.  Review the highlighted results, edit if necessary, and hit **[ COPY RESULT ]**.

---

### **UI / Design System**
* **Aesthetic:** Clean white/gray interface with high-contrast black buttons.
* **Ergonomics:** 15px rounded corners (border-radius) on all elements for a modern, tactile feel.
* **Mobile Ready:** Responsive design with specific fixes for iOS Safari text rendering.
